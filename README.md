# Python Projects

A collection of beginner Python projects built from scratch.

## Projects

### 1. Number Guessing Game
A game where the computer picks a random number and the user guesses it.
Features: hints, attempt counter, play again option.

### 2. Calculator
A calculator that performs +, -, *, /, %, //, ** operations.
Features: division by zero handling, invalid input handling.

### 3. Quiz App
A 5-question quiz with scoring system.
Features: multiple attempts per question, case-insensitive answers.

### 4. To-Do List
A task manager with add, view and remove features.
Features: invalid input handling, task not found handling.
